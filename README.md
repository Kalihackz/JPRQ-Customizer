Installation :
=============
* Clone or Download the repository.
* Unzip the zip.
* Open a terminal in the unzipped folder namely : `jprq-customizer-main`
* Now enter the following commands in the terminal :

| Commands  |
| ------------- |
| chmod +x customize.sh |
